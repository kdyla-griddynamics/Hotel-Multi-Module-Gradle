/admin/**
/hotel/rooms : provide parameter 'type' with one of the values: 'all', 'onebedroom', 'standard', 'penthouse' to show rooms of chosen type
hotel/available : no parameter needs to be provided to show rooms available for booking
hotel/properties : provide one or many parameters 'property' with values: balcony, tv, refrigerator, minibar, jacuzzi to show rooms that have chosen properties
hotel/book : provide parameter 'number' with number of the room to be booked
hotel/booked : no parameter needs to be provided to show rooms booked by logged user
